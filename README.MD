# HOOPS Visualize 3DF - Parasolid integration
This sample was developed for the following article of Tech Soft 3D Forum:<br>
https://forum.techsoft3d.com/t/hoops-visualize-3df-parasolid-integration/1674

## Environment
### SDKs in use (version)
* HOOPS Visualize 3DF (2900)
* Siemens Parasolid (34.1.153)

### Tested server OS
* Windows 10

### Build
1. Unzip HOOPS Visualize 3DF and Parasolid SDKs 
2. Locate a valid hoops_license.h in `Dev_Tools\hoops_3dgs\source\` folders of HOOPS Visualize 
3. Edit mfc_simple_Ps_VS2019.bat accoording to your environment and open mfc_simple_Ps_v142.sln using the bat file
4. Build the solutions
